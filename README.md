# A Comparative Analysis of Algorithms for Robotic Coverage Path Planning with Soft Parallel Constraints

## Overview

This repository contains the source code developed for my MSc dissertation in Robotics and Autonomous Systems at the University of Lincoln.

The project investigates the performance of graph-based coverage path planning algorithms under soft parallel movement constraints. The work focuses on generating efficient coverage paths for robotic systems operating in structured environments such as agriculture, where robots are often required to follow lawnmower-style swathing patterns.

## Research Objectives

The project compares several graph-based coverage path planning approaches and evaluates their ability to:

* Maximise area coverage
* Reduce repeated node visits
* Avoid collisions with obstacles
* Minimise constraint violations
* Generate efficient swathing patterns

The research investigates how increasing obstacle density affects algorithm performance in robotic coverage applications.

## Algorithms Evaluated

The implementation includes variations of:

* Dijkstra's Algorithm
* A* Search Algorithm
* Directed Graph Approaches
* Undirected Graph Approaches
* Soft Parallel Constraint Methods
